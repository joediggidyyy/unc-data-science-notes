# Support

## Questions or requests

Use GitHub Issues for:

- Typos / corrections
- Requests for clarification
- Requests for new topics

## Private contact

If you are not comfortable posting an issue publicly, avoid including sensitive details.

If GitHub Security Advisories are enabled for this repository, you can open a private advisory.
Otherwise, contact the repository owner via their GitHub profile.

> Note: This repository is a student-run project. The maintainers are volunteers and may not be able to respond immediately. Please be patient and respectful in your communications.
